Интернет-магазин для заказа кроссовок - React Sneaker

Stack:

npm
ReactJS + хуки (useState, useEffect, useContext + кастомный хук useCart) 
React Router
SASS
module CSS
Axios
Кроссовки брались и отправлялись по запросу с сервера mockAPI
